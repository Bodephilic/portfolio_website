Johnson Babarinde — Portfolio (single-page)

This folder contains a single-page portfolio scaffold built with plain HTML, CSS and JavaScript.

Files
- `index.html` — the single-page portfolio
- `styles.css` — styles for the site (modern blue theme)
- `script.js` — small client script to render example projects and enable smooth scrolling

Customize
- Replace the placeholder project entries in `script.js` with your real projects (live URL and repo URL).
 - The project entries are pre-filled for: `Login UI`, `Landing Page`, and `E‑commerce Website` (you can update live/repo links in `script.js`).
- Replace the contact email in `index.html` (`mailto:your.email@example.com`) with your real email.
- Add `resume.pdf` to the folder root to enable "Download Resume".

Preview
- Open `index.html` in a browser, or serve the folder with a static server (recommended so `fetch` works for images or if you later fetch external project data):

  python -m http.server 8000
  # then open http://localhost:8000/index.html

Deploy
- This is static — deploy to GitHub Pages, Netlify, or any static host. If you want, I can help deploy it to GitHub Pages or Vercel.
